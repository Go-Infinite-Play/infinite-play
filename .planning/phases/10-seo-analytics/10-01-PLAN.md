**Per DSGN-02 — Add Plausible Analytics with conversion event tracking.**

1. In `app/layout.tsx`, add the Plausible Analytics script tag inside the `<head>` section. Since Next.js App Router handles `<head>` through metadata, use a `<Script>` component from `next/script`:
   ```tsx
   import Script from "next/script"
   ```
   Add inside the `<body>` tag, before `{children}`:
   ```tsx
   <Script
     defer
     data-domain="infiniteplay.ai"
     src="https://plausible.io/js/script.js"
     strategy="afterInteractive"
   />
   ```

   Also add a global type declaration for window.plausible. Add this either at the top of layout.tsx or in a separate `types/plausible.d.ts`:
   ```typescript
   declare global {
     interface Window {
       plausible?: (event: string, options?: { props?: Record<string, string> }) => void
     }
   }
   ```

2. Create a helper function. Add to `lib/analytics.ts` (new file — but keep it tiny):
   ```typescript
   export function trackEvent(event: string, props?: Record<string, string>) {
     if (typeof window !== "undefined" && window.plausible) {
       window.plausible(event, props ? { props } : undefined)
     }
   }
   ```

3. In `components/CalendlyButton.tsx`, import `trackEvent` from `@/lib/analytics` and call `trackEvent("Calendly Click")` when the button is clicked (in the onClick handler, before or after opening Calendly).

4. In `components/ContactForm.tsx`, import `trackEvent` and call `trackEvent("Contact Form Submit")` after successful form submission (inside the success handler).

5. In `components/LeadMagnet.tsx`, import `trackEvent` and call `trackEvent("Lead Magnet Submit")` after successful email capture submission.

**Event names for Plausible goals:**
- `Calendly Click` — primary conversion
- `Contact Form Submit` — secondary conversion
- `Lead Magnet Submit` — lead capture conversion
